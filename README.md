# NutriGuide AI Lite

Веб-приложение для быстрого расчёта рациона питания под цель и бюджет.

**Что делает:**
- собирает данные через пошаговую анкету (пол, возраст, рост, вес, цель, активность, бюджет, страна, ограничения);
- считает BMR по формуле Миффлина — Сан Жеора, TDEE и целевую калорийность;
- считает БЖУ;
- генерирует меню на день и список покупок на неделю с учётом бюджета;
- оценивает примерную стоимость корзины в местной валюте;
- даёт прогноз результата на месяц.

Все расчёты происходят локально в браузере — никакие данные никуда не отправляются, никакого API не требуется.

## Стек

- **Next.js 14** (App Router) — фреймворк на базе React
- **React 18** — UI
- **Tailwind CSS** — стили
- **JavaScript** (без TypeScript — для простоты)

## Запуск локально (на компьютере)

**Что нужно установить заранее:**
1. **Node.js** (версия 18 или новее) — [скачать с nodejs.org](https://nodejs.org/)
2. **Git** — [скачать с git-scm.com](https://git-scm.com/)
3. Любой редактор кода, например **VS Code**

**Шаги:**

```bash
# 1. Перейти в папку проекта
cd nutriguide-ai-lite

# 2. Установить зависимости (один раз)
npm install

# 3. Запустить локальный сервер
npm run dev
```

После этого откройте в браузере: **http://localhost:3000**

Приложение будет автоматически перезагружаться при изменении файлов.

## Публикация на Vercel через GitHub

### Шаг 1. Создать репозиторий на GitHub

1. Зайдите на [github.com](https://github.com) и войдите в аккаунт (или создайте).
2. В правом верхнем углу нажмите **+** → **New repository**.
3. Введите имя, например `nutriguide-ai-lite`.
4. Оставьте Public (или сделайте Private — оба варианта работают с Vercel).
5. **Не** ставьте галочки на «Add README», «Add .gitignore», «Choose a license» — они уже есть в проекте.
6. Нажмите **Create repository**.

### Шаг 2. Загрузить код в GitHub

Откройте терминал в папке `nutriguide-ai-lite` и выполните по очереди:

```bash
# Инициализировать git-репозиторий
git init

# Подключить к удалённому репозиторию (замените YOUR_USERNAME на свой логин GitHub)
git remote add origin https://github.com/YOUR_USERNAME/nutriguide-ai-lite.git

# Добавить все файлы
git add .

# Сделать первый коммит
git commit -m "Initial commit"

# Переключиться на ветку main
git branch -M main

# Загрузить код на GitHub
git push -u origin main
```

Если git попросит логин/пароль — введите логин GitHub и Personal Access Token (создаётся в Settings → Developer settings → Personal access tokens → Tokens classic → Generate new token, со scope `repo`).

### Шаг 3. Подключить Vercel

1. Зайдите на [vercel.com](https://vercel.com) и войдите через **Continue with GitHub**.
2. На дашборде нажмите **Add New...** → **Project**.
3. В списке репозиториев найдите `nutriguide-ai-lite` и нажмите **Import**.
4. Все настройки оставьте по умолчанию — Vercel автоматически распознаёт Next.js.
5. Нажмите **Deploy**.

Через 1–2 минуты приложение будет доступно по ссылке вида:
`https://nutriguide-ai-lite-xxxxx.vercel.app`

Этой ссылкой можно делиться.

### Шаг 4. Обновления

Любой `git push` в ветку `main` автоматически обновляет сайт на Vercel — никаких дополнительных действий не нужно.

```bash
# Внести изменения в код, затем:
git add .
git commit -m "Описание изменения"
git push
```

## Структура проекта

```
nutriguide-ai-lite/
├── app/
│   ├── globals.css         # глобальные стили + Tailwind
│   ├── layout.js           # корневой layout (метаданные, шрифт)
│   └── page.js             # главная страница: переключение анкета ↔ результат
├── components/
│   ├── ProgressBar.js      # полоска прогресса анкеты
│   ├── Survey.js           # пошаговая анкета (7 шагов)
│   └── Results.js          # экран результатов
├── lib/
│   ├── calculations.js     # формула Миффлина — Сан Жеора, БЖУ
│   ├── menuData.js         # шаблоны блюд по бюджетам (эконом/стандарт/комфорт)
│   ├── menuGenerator.js    # сборка дневного меню и недельной корзины
│   └── prices.js           # оценка стоимости в тенге и рублях
├── package.json            # зависимости и npm-команды
├── next.config.mjs         # конфиг Next.js
├── tailwind.config.js      # палитра sage (зелёные тона), шрифт Inter
├── postcss.config.js       # связывает Tailwind с PostCSS
├── jsconfig.json           # поддержка алиаса @/...
├── .gitignore              # что не загружать в git
└── README.md               # этот файл
```
